# 微信小程序AppID配置指南

## 问题说明

您遇到的错误信息：
```
游客请注意游客模式下，调用 wx.operateWXData 是受限的, API 的返回是工具的模拟返回
```

这是因为项目当前使用的是游客模式 (`appid: "touristappid"`)，某些API在游客模式下功能受限。

## 解决步骤

### 1. 获取正式AppID

#### 方法一：注册微信小程序（推荐）
1. 访问 [微信公众平台](https://mp.weixin.qq.com/)
2. 点击"立即注册" → 选择"小程序"
3. 按照指引完成注册流程
4. 登录后进入"开发管理" → "开发设置"
5. 复制您的 AppID

#### 方法二：使用已有小程序
如果您已经有微信小程序，可以直接使用现有小程序的AppID。

### 2. 更新项目配置

1. 打开微信开发者工具
2. 在项目根目录找到 `project.config.json` 文件
3. 将其中的 `"appid": "touristappid"` 替换为您的真实AppID
4. 例如：
   ```json
   "appid": "wxa1b2c3d4e5f6g7h8"  // 替换为您的实际AppID
   ```

### 3. 重新编译项目

1. 保存配置文件更改
2. 在微信开发者工具中点击"编译"按钮
3. 重新运行小程序

## 注意事项

- **开发阶段**：可以使用测试号进行开发调试
- **生产部署**：必须使用正式的小程序AppID
- **API权限**：正式AppID可以使用完整的微信API功能
- **域名配置**：生产环境还需要配置合法域名白名单

## 常见问题

### Q: 没有微信公众号怎么办？
A: 可以直接注册小程序，不需要先有公众号

### Q: AppID在哪里找到？
A: 微信公众平台 → 开发管理 → 开发设置 → 小程序AppID

### Q: 游客模式有什么限制？
A: 主要限制包括：
- 部分API调用受限
- 无法调用用户敏感信息相关接口
- 网络请求可能受到限制
- 无法进行真机调试

## 验证配置成功

配置完成后，您应该能够：
- 正常调用所有微信API
- 不再出现游客模式提示
- 进行完整的功能测试

如有其他问题，请参考微信官方文档或联系技术支持。
